# Dynamic Pathfinding Agent

This project is a grid-based pathfinding visualizer made using Python and Pygame.
It shows how A* and Greedy Best-First Search (GBFS) algorithms find paths from a start node to a goal node.

You can change grid size, generate random mazes, add/remove walls, turn on dynamic obstacles, and see live performance results.



## Requirements

Make sure you have **Python 3** installed.

Then install Pygame:

pip install pygame


## How to Run the Project

1. Download or clone this project.
2. Open the project folder.
3. Open terminal or command prompt in that folder.
4. Run the file:


python filename.py


(Replace `filename.py` with your actual Python file name.)


## Controls

* **Left Click** → Add wall
* **Right Click** → Remove wall
* **START** → Start pathfinding
* **STOP** → Stop simulation
* **Generate Maze** → Create random obstacles
* **Dynamic ON/OFF** → Toggle moving obstacles
* **Sliders** → Change grid size and speed



## Features

* A* and GBFS algorithms
* Manhattan and Euclidean heuristics
* Adjustable grid size
* Dynamic obstacles
* Real-time performance metrics


This project helps in understanding how AI search algorithms work in a simple and visual way.
